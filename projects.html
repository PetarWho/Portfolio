Projects

If you want to participate in mine or invite me in other projects, please contact me.

Dungeon Runner

A simple game made on Unity back when my coding journey was starting. You had 4 different weapons that you have to find throughout several dungeons, as you had to kill every single enemy or boss to complete the level. It was meant for speedruns. Sadly it's all gone since I did not use Github back then.

Milk Clicker Universe

A clicker game that I was planning to finish and release on App Stores since it was meant to be a mobile game. I got tons of ideas for this game and raised my goals, which lead to demotivation. So all the ideas can still be found on DevOps and the project is currently "frozen" and kept as private on Github.

SpeedTyper

Word falling type of game, where you have to type every single word that falls, as it penalizes for each mistake. It has 4 main difficulties and other modes to play as well. Collect coins and trophies and improve your typing speed in a fun to play way. I was planning to release it this year on Steam. 

LeagueChamps

App made on C# using Windows Forms. Took me 2 hours to make and I use it to get the count of my unowned champions and the total amount of BE that I will need to buy them all. The only down side is that you have to enter all the info. But I plan on reworking it with Riot's API. Can be found on Github.

OddsCalculator

I liked playing around on different CS:GO gambling sites, so I decided to make an odds calculator to find out the most profitable boxes. Of course it works with everything not just CS:GO sites. You can copy ton of percentages and paste them. It is hosted using Github Pages.

TapVacation (internship)

With team of 8 people split to  Backend and Frontend teams. We made an Angular app for employers and employees to use for sending requests for absence to their employers as they have to explain the reason and the period of their desired absence. The employer can approve or reject their request.

Play it Wright

An automated way to claim the free game from Epic Games each week. It's written using Playwright - an end-to-end testing package that allows automation of every action that can be done to a website. And I use it to login and claim free games each week with 100 accounts. The only obstacle is Captcha..

Blender Projects

From cubes to bottles to cars to racetracks. Blender is my favorite 3d modeling software. I've made couple of models there and I can't say I'm an expert. More like beginner\intermediate. As it is so hard to master. I praise all the people that have mastered it. They are pure legends. <3

FileRevamp

FileRevamp is a simple app built in Windows Forms, that provides an easy way to manipulate file settings, such as name, extension, creation date, etc. Accepts all file types (images, text, video, audio) It works with single file or many files at once.  Download link can be found on my Github.
